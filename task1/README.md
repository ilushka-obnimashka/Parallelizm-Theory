1. Соберите проект с помощью make (предварительно выбрав способ выполнения):
   **Сборка с использованием `float`:**
   ```sh
   make task1_float
   ```
   **Сборка с использованием `double`:**
   ```sh
   make task1_double
   ```
2. После успешной сборки проекта вы можете запустить код:
   ```sh
    ./task
   ```
